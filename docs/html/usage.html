filework: Usage¶

  • Instantiate an Object:

    fw = Filework("path/to/edda_file.txt")
    ... or ...
    fw = Filework()

  • Set or re-set the path to the file:
    fw.file_path = "path/to/edda_file.txt"

  • Write To a File:

    Pass in either a string ...
    fw.write_to_file ( "The tenth is Glitnir;\nits pillars are gold\nAnd its roof with silver is set;")
    or a list.
    fw.write_to_file( ["The tenth is Glitnir;", "its pillars are gold", "And its roof with silver is set;"] )

  • Append To A File:

    Pass in either a string ...
    fw.append_to_file( "The tenth is Glitnir;\nits pillars are gold\nAnd its roof with silver is set;") )
    or a list.
    fw.append_to_file( ["The tenth is Glitnir;", "its pillars are gold", "And its roof with silver is set;"] )

  • Iterate Through a File:

    Assume the arguments passed in in the methods above have been written to a file. Each call of this method will produce one line of the file at a time.
    fw.file_path = "path/to/edda_file.txt"
    content = fw.iterate_through_file("n") # call the method ...
    print content #prints "The Tenth is Glitnir;"
    content = fw.iterate_through_file("n") # ... call the method again
    print content #prints "its pillars are gold"
    content = fw.iterate_through_file("n") # ... and again
    print content #prints "And its roof with silver is set;"
    content = fw.iterate_through_file("n") # ... and again
    print content # ...reached the end of the file, so prints "EOF"

  • Read From File:

    content = fw.read_from_file()
    print content # prints "The Tenth is Glitnir;\nits pillars are gold\nAnd its roof with silver is set;"

  • Delete A File:

    Assume that the file_path has been set as shown above. This method deletes the file that is pointed-to by file_path.
    fw.delete_file()

  • Get the results of the Method Call using the result and status members:

    Did the method call succeed or fail? Check the result:
    fw.result
    print fw.result # fw.result will either be 'OK' if successful or 'FAIL' if not.
    ... if fw.result is 'FAIL', any error message from a caught exception will be in fw.status:
    fw.status
    print fw.status # if 'FAIL', fw.status will contain a message about what happened.
